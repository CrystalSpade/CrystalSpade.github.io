# Florence County Web & Accessibility Work

During my tenure as a Web Developer with Florence County, SC, I designed and implemented internal and public-facing web systems focused on usability, accessibility, and performance.

Highlighted work includes:
- ADA accessibility toolkit implementation
- Internal Human Resources web portal
- Employee phone directory system
- Parks & Recreation eSports event website

For security and confidentiality reasons, production code and database integrations are excluded. The materials provided here demonstrate functionality, interface design, and accessibility engineering practices.

# Florence County – Portfolio Assets

This directory contains visual assets used to support portfolio demonstrations of web systems and accessibility tooling.

## Included Asset Groups

### ADA Accessibility Toolkit
Screenshots representing UI state changes including Dark Mode, High Contrast, Dyslexia font, Visual Focus, Keyboard Navigation visibility, and draggable control panel behavior. Demonstrates responsive and user-controlled accessibility features.

### eSports Website
Desktop layout captures showing public-facing event site structure, navigation hierarchy, and accessibility integration.

### Internal Systems
Interface previews of departmental tools such as Human Resources and Phone Directory layouts.

## Notes
All materials are provided for portfolio demonstration purposes. No sensitive or private data is included. Images are optimized for web display.

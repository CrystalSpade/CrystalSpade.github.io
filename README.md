# Web Systems & Accessibility Portfolio

This repository contains the source code and supporting assets for my professional web development portfolio. The site highlights publicly demonstrable projects involving accessibility tooling, responsive design, and structured web system implementation.

---

## Overview

The portfolio focuses on:

- Front-end web development
- Accessibility-first UI engineering
- Responsive layout implementation
- Public-facing and internal system interfaces
- Modular, reusable UI components

The goal of this portfolio is to demonstrate practical implementation of production-style web systems with emphasis on usability, structure, and accessibility compliance.

---

## Featured Projects

### ADA Accessibility Toolkit
A customizable accessibility layer implemented on a public-facing web interface.  
Features include:

- Dark Mode
- High Contrast mode
- Font size adjustments
- Dyslexia-friendly font toggle
- Visual focus enhancements
- Keyboard navigation visibility
- Draggable control panel interface
- Responsive mobile support

Video demonstrations are embedded via YouTube for optimized delivery.

---

### Florence County eSports Website
Public-facing event platform demonstrating:

- Structured navigation hierarchy
- Hero layout implementation
- Quick links section
- Responsive behavior
- Accessibility integration

A PDF layout capture is included for structural reference.

---

### Internal System Interfaces
UI demonstrations representing structured departmental layouts, including:

- Human Resources portal
- Employee phone directory interface

These examples highlight interface organization, navigation clarity, and scalable layout practices.

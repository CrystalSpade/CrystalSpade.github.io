Project Highlights

ADA Accessibility Control System

Built an interactive accessibility control widget for Florence County's public-facing website, enabling users to customize display settings to meet their individual accessibility needs.

  • UI state persistence across page navigation
  • Theme rendering (high-contrast, dark mode)
  • Keyboard enhancement controls
  • WCAG 2.1 AA compliant

WCAG Compliance Improvements

Audited and remediated accessibility issues across Florence County's public services web presence to meet WCAG 2.1 Level AA standards.

  • Color contrast audits and remediation
  • Semantic HTML restructuring
  • ARIA landmark and label implementation
  • Keyboard navigation and focus management

Internal HR Portal

Developed an internal HR portal and employee directory system to support county staff operations and reduce administrative overhead.

  • Employee directory with search functionality
  • Role-based access control
  • Responsive design for desktop and mobile
  • Cross-browser compatibility

Front-End Performance Optimization

Optimized performance and UI behavior across desktop and mobile platforms to improve page load times and user experience for county residents.

  • Asset optimization and lazy loading
  • Mobile-first responsive layouts
  • Cross-browser stability improvements
  • Stakeholder collaboration and delivery
